Description:

This is a grades table that I created. There are eight students with their exam grades, and I sorted their names in alphabetical order using the sort function and calculated the averages grades. I also created a grading scheme as a reference with the grade table.

Screenshot

Named ranges:

"grades" from A14 to A18; "ltr" from B14 to B18.

Formulas:

1. Grade =AVERAGE(B3:E3) [the selection value is according to every student's exam 1 to final] 2. Ltr_Grade =LOOKUP(F3,grades,ltr) [the selection value like F3 changes according to every student's Grade]
