Description:

I created the first table using the convert function to convert Celsius to Farenheit and Kelvin. Then I used the expand function to expand the table to 40. The second table was then done using VLOOKUP function.

Screenshot

Named ranges:

"Fahrenheit" from C2 to C42; "Kelvin" from D2 to D42.

Formulas:

Table A: 1. Fahrenheit =CONVERT(B2,"cel","fah") 2. Kelvin =CONVERT(B2,"C","K") Table B: 1. F° =VLOOKUP(G2,B:C,2,FALSE) 2. K° =VLOOKUP(G2,B:D,3,FALSE)
